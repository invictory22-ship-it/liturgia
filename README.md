# Голосовий служебник — Літургія Іоана Золотоустого

Веб-додаток, що слухає голос священника під час богослужіння й підсвічує
поточне місце в тексті.

## Як запустити на своєму комп'ютері
Подвійний клік на **`start.bat`** → відкриється Chrome за адресою
`http://localhost:8000`. Тільки так працює мікрофон локально.
(Потрібен встановлений Python — у тебе вже є.)

Якщо хочеш просто почитати текст без мікрофона — можна двічі клікнути
`index.html`.

## Структура папок
- **`index.html`** — сам додаток (програма). Текст тут не правимо.
- **`liturgy-data.js`** — ТЕКСТ Літургії блоками. Це той файл, який вичитуємо й правимо.
- **`liturgy-source.txt`** — авторитетне джерело тексту (з нього зроблено розмітку).
- **`CLAUDE.md`** — опис проєкту для Claude.
- **`CORRECTIONS.md`** — правки від священника.
- **`pdf/`** — PDF інших богослужінь (див. pdf/README.txt).
- **`archive/`** — старі версії додатку (не використовуються).

## Онлайн-версія (для телефона)
Сайт: **https://invictory22-ship-it.github.io/liturgia/**
Оновлюється сам через GitHub Pages після кожного коміту (1-2 хв).
